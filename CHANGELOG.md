# Changelog

## v1.0.0 — 2026-03-13

Initial release.

- 18+ statusline elements (battery, model, context %, burn rate, session cost, streak, clock, git branch, and more)
- Sound hooks for tool errors, stops, and notifications
- Interactive TUI configurator (`configure.py`) with live preview
- Obsidian integration
- Session git cleanup hook
- Per-type notification hooks with iTerm2 OSC 9 support
- npm distribution via `fantasy-claude` package
